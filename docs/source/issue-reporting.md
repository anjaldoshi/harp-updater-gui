# Issue Reporting

Use this page when you hit errors, unexpected behavior, or setup problems.

## Before filing an issue

Please collect:

- OS version (Windows/macOS/Linux)
- App version or commit SHA
- HarpRegulator version
- Device model/type and connection method
- Exact steps to reproduce
- Expected vs actual behavior
- Relevant activity log output
- Error dialogs or terminal output (screenshots if helpful)

## Where to file

Open an issue on GitHub:

- [https://github.com/AllenNeuralDynamics/harp-updater-gui/issues](https://github.com/AllenNeuralDynamics/harp-updater-gui/issues)

- Search existing GitHub issues first.
- Add reproducible steps and logs for faster triage.
